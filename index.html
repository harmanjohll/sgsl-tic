How it works

Your hand movements are tracked in real-time using MediaPipe's hand pose model. Each recording captures 21 hand landmarks across every frame, encoding both hand shape and movement. These are normalized and stored as feature vectors for sign recognition. The more samples contributed, the better the recognition becomes.

About the recognition

Signs are recognized by comparing hand movement sequences using Dynamic Time Warping (DTW) — an algorithm that measures similarity between temporal sequences, even if performed at different speeds. Each hand pose is normalized (position, scale, rotation invariant) and converted to a 59-dimensional feature vector per frame. Top matches are shown with confidence scores.
